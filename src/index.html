Documentação detalhada

Flexibilidade

  • Escreva o seu template em um arquivo HTML
  • Escreva o seu template em um arquivo Javascript
  • Use JSX no seu arquivo Javascript
  • Faça o seu template com Javascript puro usando virtual nodes

Essa flexibilidade faz com que desenvolvedores habituados com React e Angular migrem com facilidade para o Vue, uma vez que eles encontrarão estruturas familiares.

Direito de escolher

  • Template
    • html binding
    • jsx
    • jade
  • Component class
    • es5
    • es6 / es7
    • es6 + flow
    • TypeScript

Desvantagens

  • A integração com Typescript na v2.x.x não é boa
  • Não é possível levar com tanta facilidade os conhecimentos do framework para o mundo mobile como é com o React/React Native

Propriedades computadas e Observadores

Watch


var vm = new Vue({
  el: '#demo',
  data: {
    firstName: 'Foo',
    lastName: 'Bar',
    fullName: 'Foo Bar'
  },
  watch: {
    firstName: function (val) {
      this.fullName = val + ' ' + this.lastName
    },
    lastName: function (val) {
      this.fullName = this.firstName + ' ' + val
    }
  }
})

Computed


<div id="demo">{{ fullName }}</div>
                

var vm = new Vue({
  el: '#demo',
  data: {
    firstName: 'Foo',
    lastName: 'Bar'
  },
  computed: {
    fullName: function () {
      return this.firstName + ' ' + this.lastName
    }
  }
})

Computed Setter


//...
computed: {
  fullName: {
    // getter
    get: function () {
      return this.firstName + ' ' + this.lastName
    },
    // setter
    set: function (newValue) {
      var names = newValue.split(' ')
      this.firstName = names[0]
      this.lastName = names[names.length - 1]
    }
  }
}
//...

Event & Key Modifiers


<!-- the click event's propagation will be stopped -->
<a @click.stop="oThis"></a>
<!-- the submit event will no longer reload the page -->
<form @submit.prevent="onSubmit"></form>
<!-- modifiers can be chained -->
<a @click.stop.prevent="doThat"></a>
<!-- also available .tab, .delete, .esc, .space, ...-->
<input @keyup.enter="submit">
